# MEP 機水電設計圖說 自主檢查系統

**MEP Drawing Self-Inspection & Confirmation System**

機電（M）、給排水（P）、消防、空調 設計圖說送審前自主核對工具。

## 功能特色

- **8 大系統** 分類核對：電力（含7個子系統）、弱電、給水、排水、消防、空調、通風、避雷接地
- **電力系統深化**：動力 / 照明 / 系統單線升位圖 / 太陽能 / 大樣圖 / 外牆燈 / 緊急發電機
- **117 個核對項目**，A/B/C 三級嚴重性分類
- **三態勾選**：✓ 符合 / ✗ 有問題（自動展開備註欄）/ — 不適用
- **即時進度追蹤**：每個子系統進度條 + 整體完成百分比
- **法規依據引用**：每項均標示台灣法規 / 業主規範（00702 CSI MasterFormat）
- **自動儲存**：每次操作寫入瀏覽器 localStorage，重開不遺失
- **冷啟動恢復**：開啟時偵測上次作業，選擇繼續或重新開始
- **列印 / PDF 輸出**：
  - 頂部統計摘要（總計 / 符合 / 有問題 / 不適用 / 待核對）
  - 底色辨識（綠色=符合 / 紅色=有問題 / 黃色=待核對 / 灰色=不適用）
- **簽署欄位**：每個子系統獨立的設計工程師 / 複核工程師 / 日期
